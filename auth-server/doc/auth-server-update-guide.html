‫‫مستند بروزرسانی و پیکربندی مولفه Auth-Server  از نسخه 4.0.2  به نسخه 5.0.0

بروزرسانی دیتابیس

فایل update4.0.2-to-5.0.0.sql را از بسته Update-DB.tar بر روی سرور MYSQL کپی کنید و سپس دستور زیر را  بر روی سرور MYSQL اجرا کنید:

mysql --verbose -u${DBA_USER} -h ${DBSERVER} -p${DBA_PWD} ${DBNAME} < update.sql > auth-server.log 2>&1
DBSERVER=localhost آدرس ای پی سرور MYSQL
DBA_USER=root نام کاربری کاربر مدیرMYSQL
DBA_PWD رمز عبور کاربر مدیر MYSQL

اگر خطایی باشد در فایل auth_server.log می توانید مشاهده کنید.

فایل permissions-auth.json را از بسته با استفاده از ابزار permission-importer.sh به دیتابیس وارد کنید.

برای این نسخه باید audit-server نسخه 5.0.0 را نصب کنید.

سپس سیستم را ری استارت کنید.
