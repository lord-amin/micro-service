‫‫مستند راهنمای  permission-importer

برای اینکه سطح دسترسی ها را به auth-server معرفی کنیم باید فایل json دسترسی های سرویس مورد نظر را بوسیله ابزار permission-importer.sh به دیتابیس auth-server وارد کنیم.

این ابزار به 2 صورت قابل استفاده است:

1- اگر دستور زیر را اجرا کنیم 

./permission-importer.sh -a update -f {/path/to/json/file} -s {serviceName} -l error

دسترسی های موجود در فایل به دیتابیس اضافه می شوند . اگر دسترسی خاصی در دیتابیس نباشد اضافه شده در غیر اینصورت فیلد های آن بروز می شوند.قابل ذکر است که serviceName می تواند هر رشته ای باشد که معمولا مقدار معناداری برای نشان دادن نام سرویس مورد نظر است.

2- اگر دستور زیر را اجرا کنیم

./permission-importer.sh -a delete -s {serviceName} -l error

تمام دسترسی های موجود که نام سرویس آنها برابر با serviceName است ، از  دیتابیس پاک می کند.
